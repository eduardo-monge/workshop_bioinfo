# Guia Prático - Dia 2: Processamento e mapeamento dos arquivos fastq
Bem-vindos ao Dia 2! Hoje continuaremos nosso tutorial de bioinformática entrando na parte mais divertida: colocar a mão na massa com os dados reais. Nosso fluxo de trabalho começará com o Controle de Qualidade (QC) das sequências para diagnosticar possíveis falhas. Em seguida, faremos a "limpeza" (Trimming) para eliminar os adaptadores e bases com erros de leitura. Finalmente, com os arquivos limpos e de alta confiança, faremos o mapeamento dessas sequências contra o nosso genoma de referência."
