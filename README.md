📂 Workshop de Bioinformática: Da Sequência ao VCF
Bem-vindo ao repositório central do nosso workshop! Este espaço contém todo o material teórico, roteiros práticos e códigos necessários para realizar o fluxo completo de análise de variantes genéticas.

🎯 Objetivos do curso
1. Dominar a linha de comando Linux voltada para genômica.
2. Implementar boas práticas de organização e reprodutibilidade.
3. Realizar o controle de qualidade (QC) e alinhamento de sequências.
4. Executar a chamada de variantes (Variant Calling) utilizando abordagens baseadas em bases e haplótipos.
5. Filtrar e normalizar arquivos VCF.
