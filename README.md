# 📂 Workshop de Bioinformática: Da Sequência ao VCF

Bem-vindo ao repositório central do nosso workshop! Este espaço contém todo o material teórico, roteiros práticos e códigos necessários para realizar o fluxo completo de análise de variantes genéticas.

## 🎯 Objetivos do curso
* Dominar a linha de comando **Linux** voltada para genômica.
* Implementar boas práticas de **organização e reprodutibilidade**.
* Realizar o controle de qualidade (QC) e alinhamento de sequências.
* Executar a chamada de variantes (**Variant Calling**) utilizando abordagens baseadas em bases e haplótipos.
* Filtrar e normalizar arquivos **VCF**.
